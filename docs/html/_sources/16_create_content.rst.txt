CREATE CONTENT
==============

Convert movies to h.264 * .mp4 files
************************************


.. image:: _images/16_handbrake.png





I recommend to use https://handbrake.fr for encoding videos.

A lot of people still use Adobe media encoder, which is, in my opinion, unusable bloat... but use whatever you think creates the best results.

Using Handbrake:
*****************

Make sure to disable Auto Cropping in the "Dimensions" tab.


In the  "Video" tab,  select:

Framerate: same as source



.. image:: _images/16_framerate.png


.. image:: _images/16_dimensions.png

If you get videos with ending .m4v, disable the check mark in settings:


.. image:: _images/16_handbrake_prefs.png
